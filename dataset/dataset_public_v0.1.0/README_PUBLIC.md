# LUM-PE Public Reference Dataset v0.1.0 (SYNTHETIC)

This is a **public, general-purpose, synthetic** dataset intended to:
- Provide a **Hello World at scale** for LUM-PE (45 fields).
- Enable **end-to-end reproducibility** of bundle generation and hashing.
- Serve as a **public reference** for documentation, onboarding, and CI smoke tests.

## What this dataset is
- A **synthetic** dataset (not collected from real studies).
- Designed to be **non-sensitive** and publishable.

## What this dataset is NOT
- It is **not** an empirical validation of LUM-PE.
- It does **not** justify claims like AUC/ECE for real domains.

## Reproducibility
- Seed: 2602026
- Generated by: `generate_public_dataset.py`
- Main table: `evidence_set_table_public_45.csv`
- Bundles: `bundles_public/` (one per field, auditable with SHA-256 footprint)
