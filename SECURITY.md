# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅ Yes    |

## Reporting a Vulnerability

If you discover a security vulnerability in LUM-PE, please report it privately by emailing:

**jules_pintor@outlook.com**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 72 hours and work to address the issue promptly.

## Gaming / Adversarial Concerns

LUM-PE includes a **BLACK state** and shadow index specifically designed to detect gaming of the scoring system. If you discover a bypass that produces a fraudulent GREEN state, this is treated as a critical vulnerability. Please report it through the same channel above.
